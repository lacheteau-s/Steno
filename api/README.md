# Steno - API

## Requirements

| Component | Version |
|-----------|---------|
| [.NET SDK] | 9.x |

## Getting started

### Using the `dotnet` CLI

```sh
cd src/Steno.Api
dotnet run
```

### Using VS Code (debugging)

1. Go to the `Run and Debug` view.
2. Select `Steno.Api` from the dropdown.
3. Press `F5` or click the :arrow_forward: button
